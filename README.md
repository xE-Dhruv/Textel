# Textel

Convert images into ASCII art directly in your browser.


</div>



## About

Textel is a lightweight web application that converts images into ASCII art.

All processing happens locally in your browser, ensuring fast performance and complete privacy.


## Features

- Image to ASCII conversion
- Instant preview
- Download ASCII output
- Responsive design
- No server required
- Privacy-friendly

## How It Works

1. Upload an image.
2. Textel analyzes image brightness values.
3. Pixels are mapped to ASCII characters.
4. The generated ASCII art is displayed instantly.
5. Download the result.

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Canvas API
